# AMD OpenHands Playbooks

This repository contains OpenHands tutorials for AMD systems. Each tutorial is
kept in its own directory with its own `README.md`, metadata, platform notes,
and assets.

## Tutorials

| Tutorial | Description |
| --- | --- |
| [Getting Started with OpenHands and a Local LLM](openhands-getting-started/README.md) | Install Agent Canvas, connect a local Lemonade model, and run your first OpenHands coding agent task. |
| [Build a GitHub-to-Slack Development Digest](github-slack-development-digest/README.md) | Configure Agent Canvas, GitHub MCP, Slack MCP, and a scheduled OpenHands automation that posts a development digest. |
